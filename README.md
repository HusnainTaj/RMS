# RMS
An online Restaurant Management System in .Net Core MVC

# Running
- Clone the repository
- Open `RMS.sln` in Visual Studio
- Open Package Manager Console (`Ctrl + /` then `Ctrl + .`)
- Run `Update-Database` command
- Run the project (`F5`)
